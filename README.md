# Job Application Plugin

A cross-platform job application assistant that helps users fill out applications without losing their information.

## Features
- Works on phones, tablets, Chromebooks, and laptops
- Saves user data like name, address, phone number, email, work history, and resume
- Allows editing and updating of saved information
- Supports multiple answer options for repeated questions
- Prevents data loss from glitches, refreshes, or form resets
- Opens pop-ups or external application pages in a new tab without losing progress
- Helps users complete multi-step job applications smoothly

## Goal
To reduce repetitive typing and make job applications faster, easier, and more reliable across all devices.
